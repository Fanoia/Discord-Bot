# Fanoia Discord Bot

This bot runs off of node and docker. You can only run this bot via docker.

Use docker-compose up -d in the root dir of this repo after making a .env containing your token.

# Commands

/collab

/rps @{user}

/rpsmenu

/nm [color] [string]

/twitchlink [Twitch] [code]
